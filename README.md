### サービス概要
ポモドーロタイマーがついたデイリーToDoタスク管理アプリ

### 解決したい課題
ポモドーロとToDoタスク管理ができて、かつWebブラウザで完結するサービスが少ない。
またあってもデザイン性が低く、女性向けの物が少ない。
